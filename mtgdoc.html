This is the root of mtgdoc.link

MTGDoc.link is a redirection service providing simple shorthand links to the most up-to-date versions of various rules resources for Magic: the Gathering. It's part of the Academy Ruins project.

You can use any of the following resource names as either a path segment or a subdomain. For more information, see the Academy Ruins API docs.

Currently supported official documents:

  • CR
  • MTR
  • IPG
  • JAR

Other supported unofficial links:

  • Annotated IPG [AIPG]
  • Annotated MTR [AMTR]
  • WPN Rules & Documentation Page [WPN]
